# Mongodb Commerce

## Sobre o projeto:

Neste projeto, foram colocados em prática os conceitos acerca do **MongoDB** criando querys para a resolução de 32 desafios utilizando o banco de dados `commerce`, o qual contém dados do **McDonald's** como ingredientes, valores nutricionais e dados fictícios de vendas. Todos os arquivos dos desafios estão contidos na pasta `challenges`.
